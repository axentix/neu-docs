Neu-Axentix Builder

This builder allow you to choose theme colors you want to add to Neu-Axentix.
After choosing and adding them, give them their classnames, and you'll be able to download the css file.
This file will provide you everything you need to use these new colors with our neumorphic shadow classes with the color names you have set before.
